# Digit Recognition Model

This is a Convolutional Neural Network (CNN) trained from scratch to recognise handwritten digits. It was built without any pretrained weights — just a simple CNN architecture trained end-to-end on the MNIST dataset.

MNIST was chosen because it is a well-known benchmark dataset for digit recognition, containing 70,000 labelled images of handwritten digits (0–9). It is small enough to train quickly and reliable enough to get good results without needing a lot of compute.

The model achieved 98.99% test accuracy after 5 epochs on the MNIST dataset.

The training script is in this same folder. Run it to train the model and save the weights.
